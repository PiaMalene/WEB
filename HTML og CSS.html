Hva er HTML?

  • HTML er et formatteringsspråk som hjelper med å strukturere og presentere informasjon på websider.
  • HTML står for Hypertext markup language.
  • Før brukte man HTML til både struktur og presentasjon, men nå bruker man det kun til struktur.
  • Slik får du frem en nettwsside på pcen din:
    • Du får fram en nettside ved at nettleseren din sender en forespørsel til en server om den kan sende index filen til den siden du vil se. Deretter sender serveren index filen til nettleseren din i html kode som igjen blir omgjort til en nettside du kan se på monitoren din.
