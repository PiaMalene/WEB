Hva er HTML?

- HTML er et formatteringsspråk som hjelper med å strukturere og presentere informasjon på websider.
- HTML står for Hypertext markup language.
- Før brukte man HTML til både struktur og presentasjon, men nå bruker man det kun til struktur.
- Slik får du frem en nettwsside på pcen din:
- Du får fram en nettside ved at nettleseren din sender en forespørsel til en server om den kan sende index filen til den siden du vil se.
- Deretter sender serveren index filen til nettleseren din i html kode som igjen blir omgjort til en nettside du kan se på monitoren din.

    CSS

    Hva er CSS

    CSS står for Cascading Styling sheet og handler om semantikk på web. CSS gjør det mulig å skille mellom presentasjon og sturktur på en nettside.
    CSS har mange fordeler ved seg blant annet at man muliggjør design som ikke hadde vært mulig med bare HTML.
    Man sikrer også at man får et mer helhetlig design.
    Det fører også til mindre arbeid og mindre filer som igjen gjør nettsiden raskere.
    Det gjør også at man man for en responsiv nettside som kan leses på ulike platformer og er optimalisert for disse.

    Oppbyggning

    CSS er byggd opp I et hierarkisk måte, hvor de minste er størst elelr gjeldende. Som for eksemplel at h1 er rød overstyrer
    at body er blå. Cascade står for foss eller fallende og det vil si at man har forskjellige trinn I fossefallet. Første
    er kanskje body, andre trinn er section også tilslutt kan man ha h1 eller p. Det siste trinnet overstyrer det forrige.

    Du har tre muligheter på å bruke CSS. Du kan enten velger internt, eksternt eller inline.

    CSS syntax

    skrift

    Font-size angir størrelsen på teksten. Du kan oppgi den i prosent . Font-family angir hvilken skrifttype du skal bruker.
    Font-style angir stilen på teksten og font-weight angir tykkelsen på skriften.

    CSS Lenker

    Fire lenkestadier:

    • a:link – Vanlig lenketekst som ingen har klikket på enda
    • a:visited – En lenke etter at man har klikket på den
    • a:hover – Når man holder musen over lenken
    • a:active – Når man klikker på lenken
